Khái niệm:: [Phân tích](../../../%CE%9E%20Kh%C3%A1i%20ni%E1%BB%87m/Ph%C3%A2n%20t%C3%ADch.md)

| Loại phân tích         | Trả lời cho câu hỏi                    | Phương pháp                                                                     |
| ---------------------- | -------------------------------------- | ------------------------------------------------------------------------------- |
| Mô tả (description)    | Chuyện gì đã xảy ra?                   | Thống kê mô tả, phân đoạn dữ liệu, thị giác hóa dữ liệu                         |
| Chẩn đoán (Diagnostic) | Tại sao chuyện này lại xảy ra?         | Khoan sâu dữ liệu, khám phá dữ liệu, phân tích tương quan, phân tích bất thường |
| Dự báo (predictive)    | Tiếp theo thì chuyện gì có thể xảy ra? | Mô hình hồi quy, phân tích dãy thời gian, học máy, cây quyết định               |
| Đề xuất (prescriptive) | Làm gì để đạt kết quả tốt nhất?        | Tối ưu hóa, mô phỏng, phân tích scenario, lý thuyết trò chơi                    |
