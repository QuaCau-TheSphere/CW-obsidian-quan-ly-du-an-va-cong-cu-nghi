Các buổi thảo luận này hy vọng sẽ đem lại cho bạn những hiểu biết sâu sắc hơn về

Mỗi buổi sẽ có 3 lớp:
- lớp ý tưởng: cho bạn biết từ khoá, là sự kết hợp giữa what và why
- lớp kỹ thuật: cho bạn thực hành, how
- lớp văn hóa, triết học: what's more
Buổi 1: file là gì? Và sự tự trị dữ liệu của bạn liên quan gì đến nó? 

Buổi 2: markdown và tài liệu động. Công cụ nghĩ
